
Level 1: The Basics (Filtering and Selecting)
1. (Tienda) Find specific products by price. Write a query to show all data for products that cost more than 100 €. Hint: You need a comparison operator.
select * from productos where Precio > 100;
2. (Liga) Search for text patterns. Find all data for teams where the website (web) contains the word "basket". Hint: Use the % wildcard.
select * from equipo where web like '%basket%';
3. (Tienda) Filter by multiple conditions. List all clients who live in either 'Madrid' or 'Barcelona'. Hint: You can use OR logic.
select * from Clientes where Ciudad = 'Madrid' or Ciudad = 'Barcelona';
4. (Liga) Filter by date parts. Find all matches (partido) that were played specifically in the month of February (Month 2). Hint: Use a function to pull the month out of the fecha column.
select * from partido WHERE EXTRACT(month FROM fecha) = 2;

5. (Tienda) Sorting data. List all products, but order them by price from highest to lowest. Hint: ASC is default, what is the opposite?

6. (Liga) Cleaning up text. Show a list of team websites, but replace the text "http://" with nothing (effectively removing it). Hint: Use the REPLACE function.
select replace(web, 'http://', '') from equipo;
7. (Tienda) Uppercase conversion. Show a list of all client names converted entirely to uppercase letters. Hint: There is a function specifically for capital letters.
select upper(nombre) from Clientes

9. Answer: SELECT Pais, COUNT(Id_fab) AS Total_Fabricantes FROM Fabricantes GROUP BY Pais;
10. Answer: select equipo, avg(salario) from jugador group by equipo
11. Answer: SELECT Id_fab, COUNT(Id_producto) AS Total_Productos FROM Productos GROUP BY Id_fab HAVING COUNT(Id_producto) > 1;
12. Answer: select equipo, max(salario), min(salario), max(salario) - min(salario) as salario_dif from jugador group by equipo;
13. Answer:
SELECT N_pedido, Estado,
CASE
WHEN Estado = 'Completado' THEN 'Finalizado'
WHEN Estado = 'Cancelado' THEN 'Anulado'
ELSE 'En Gestión'
END AS EstadoPersonalizado
FROM Pedidos;
14. Answer: SELECT N_pedido, Fecha_pedido, SYSDATE - Fecha_pedido AS Dias_Transcurridos FROM Pedidos;
